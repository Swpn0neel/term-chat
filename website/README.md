# TermChat (Website)

TermChat is a premium, real-time messaging ecosystem designed specifically for developers who live in the terminal. It combines the speed of a CLI with the feature set of a modern social platform, including secure authentication, group messaging, AI-powered assistance, and lightning-fast file transfers.

---

## Quick Start

Get the documentation site running locally in seconds.

```bash
# 1. Install dependencies
$ bun install

# 2. Start the development server
$ bun dev

# 3. Open your browser
local: http://localhost:3000
```

# termchat-cli

A sleek, real-time CLI chat application built with React, Ink, and Prisma. Chat with friends directly from your terminal!

## Features

- **Terminal-Based UI**: Beautifully rendered using `ink`.
- **Authentication**: Secure login and registration.
- **Real-time Chat**: Instant messaging with online/offline status indicators.
- **AI Ready**: Support for Google Generative AI integrations.
- **Rich Aesthetics**: Vibrant colors and smooth transitions.

## Installation

Install it globally via npm:

```bash
npm install -g termchat-cli
```

## Setup

`termchat` is a zero-config CLI tool. Just install and run!

```bash
termchat
```

## Development

If you want to contribute or run it locally:

1. Clone the repo:
   ```bash
   git clone https://github.com/swapnoneel/cli-chat-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` file with `DATABASE_URL`.
4. Run in dev mode:
   ```bash
   npm run dev
   ```
